Habilidades/Competências

Estas são as habilidades e competências que possuo até o momento:

HTML, CSS e Javascript

A tríplice coroa do desenvolvimento front end.

Tailwind CSS

Um framework CSS baseado em utillity classes que tem como prioridade a facilidade de customização.

React

O React é a biblioteca mais popular do JavaScript, sendo baseada em componentes e é muito utilizada para construir interfaces de usuário (UI).

Next.js

Next.js é uma estrutura de desenvolvimento front-end React que permite funcionalidades como renderização do lado do servidor e geração de sites estáticos.

Typescript

TypeScript é uma linguagem de programação de código aberto desenvolvida pela Microsoft. É um superconjunto sintático estrito de JavaScript e adiciona tipagem estática opcional à linguagem.

Git + GitHub

Esta combinação há muito é utilizada para o versionamento e CI/CD e continua sendo uma ótima dupla.
